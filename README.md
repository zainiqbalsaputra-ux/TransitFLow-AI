# TransitFlow AI

AI-powered public transportation optimization and passenger demand forecasting system designed to reduce congestion in Jakarta through data-driven fleet allocation and transit planning.

## Features

- Passenger Demand Forecasting
- Peak Hour Prediction
- Fleet Requirement Estimation
- Route Optimization
- Schedule Optimization
- Crowding Reduction Simulation
- Interactive Decision Support Dashboard

## Data Sources

- Passenger Tapping Data
- Fleet GPS Data
- Route Information
- Station Occupancy Data
- Schedule Data
- Traffic Data

## Technology Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Streamlit
- Google Colab
